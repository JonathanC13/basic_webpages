## Additional packages
1. react-icons (https://www.npmjs.com/package/react-icons)
    > npm install react-icons --save

2. Easy Peasy Redux
    > npm install easy-peasy

## To run
1. Install dependencies
    > npm install

2. start json-server (Didn't need a db, but wanted to write fetching)
    > npm run server

3. run the app
    > npm start
