## Additional packages
1. react-router (https://reactrouter.com/)
    > npm install react-router-dom -save

2. abort-controller (https://www.npmjs.com/package/abort-controller)
    > npm install abort-controller -s

## To run
1. Install dependencies
    > npm install

2. run the app
    > npm start
